## &nbsp;about

I'm a CSE graduate from Comilla University working at the intersection of **machine learning**, **natural language processing**, and **cybersecurity** — with a particular interest in low-resource language modeling and intrusion detection. My recent work, *Hybrid Framework with Feature Selection and Explainable AI for IoT Intrusion Detection*, was published at **IEEE RAAICON 2025** (MIST, Dhaka). On the side, I work with industrial automation — PLCs, SCADA, ladder logic.

## &nbsp;featured paper

<br/>

<div align="center">

#### Hybrid Framework with Feature Selection and Explainable AI for IoT Intrusion Detection

<sub>4th IEEE International Conference on Robotics, Automation, AI, and IoT &nbsp;·&nbsp; RAAICON 2025</sub>
<br/>
<sub>Military Institute of Science and Technology, Dhaka &nbsp;·&nbsp; 27–28 Nov 2025</sub>

</div>

<br/>

> A hybrid framework integrating **Particle Swarm Optimization** for feature selection, a **Convolutional Neural Network** for representation learning, and **XGBoost** for classification on the RT-IoT2022 dataset. **SHAP** provides global and local feature attribution. The proposed model achieves **98.46%** accuracy, surpassing classical and standalone deep-learning baselines.

<br/>

<table>
<tr>
<td valign="top" width="33%">

<sub>01 — feature selection</sub>

#### pso wrapper

Particle-swarm optimisation reduces the feature space from **83 → 20** features, validated against Pearson-correlation pruning to remove multicollinear pairs.

</td>
<td valign="top" width="33%">

<sub>02 — model</sub>

#### cnn → xgboost

A CNN encoder learns dense embeddings from selected features; XGBoost classifies them — combining representational depth with robust decision boundaries.

</td>
<td valign="top" width="33%">

<sub>03 — interpretability</sub>

#### shap attribution

Global and local SHAP values reveal that **payload statistics** and **TCP flag counts** dominate the model's attack-class predictions.

</td>
</tr>
</table>

<br/>
